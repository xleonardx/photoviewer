How it works

1. Publish your folders

Share each Google Drive folder as "Anyone with the link" so the static site can read the images.

2. Map short keys

Update data/folders.json so friendly keys like summer-2026 point to real folder IDs.

3. Deploy on GitHub Pages

Push the site to GitHub, enable Pages, and restrict your Google API key to your GitHub Pages domain.
